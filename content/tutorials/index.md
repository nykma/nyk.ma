正文里使用的按键表达法：
</summary>
<p class="details">

`C-g`
: `Ctrl + g`

`C-x C-s`
: 按下 Ctrl，按 x，按 s，松开 Ctrl

`C-c C-e l o`
: 按下 Ctrl，按 c，按 e，松开 Ctrl，按 l，按 o

`C-M-b`
: 按下 Ctrl 和 Alt (Meta)，按 b，松开 Ctrl 和 Alt

`C-h P org RET`
: `Ctrl+h` ，然后按大写 `P` （ `Shift + p` ），然后打入 `org` ，回车
</p>
